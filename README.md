\# Sistema de Leilões



\## 📌 Sobre o projeto

Sistema desenvolvido em Java para gerenciamento de produtos em uma casa de leilões.



\## ⚙️ Funcionalidades

\- Cadastro de produtos

\- Listagem de produtos

\- Controle de status (À venda / Vendido)



\## 🛠️ Tecnologias

\- Java

\- MySQL

\- JDBC
